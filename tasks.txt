🧑‍💻 Developer A: Infrastructure, Authentication, and AI
This developer will focus on the core foundation of the app, security, and integrating the smart AI features.

Server Setup: Configure the Express server using strictly TypeScript and connect it to a local MongoDB (must be secured with a username and password, no cloud services like Atlas).

Authentication API: Implement secure registration and login, including external providers (Google/Facebook) and a JWT mechanism with Refresh Tokens.

React Infrastructure: Set up the React project, routing, and global state management to remember the logged-in user.

Auth Screens UI: Build the login and registration screens, ensuring proper form input validation.

User Profile API & UI: Create the backend and frontend for the user profile screen. This includes displaying their details, fetching their specific posts, and allowing the logged-in user to edit their username and profile picture.

AI Integration: Build the backend logic and frontend interface to intelligently analyze content or implement free-text search using an AI model (Gemini, ChatGPT, etc.), ensuring API rate limits are respected.

Auth & Profile Tests: Write Jest unit tests for all backend routes related to authentication and users.

🧑‍💻 Developer B: Content, Social Features, and Documentation
This developer will focus on the core social mechanics of the app—sharing content, interacting with others, and documenting the API.

Post Management API: Develop the CRUD operations for posts, ensuring uploaded images are saved locally on your server (not in the DB and not on external hosting). Implement a paging mechanism for fetching posts.

Comments & Likes API: Create efficient database schemas and backend routes for adding/deleting comments and toggling likes on posts.

Swagger Documentation: Set up Swagger and document the entire API (including Developer A's routes).

Main Feed UI: Build the main feed to display posts with scrolling pagination (no separate pages) and show the total counts of likes and comments on each post.

Post Creation & Editing UI: Create the forms allowing users to upload text and images, and give the author the ability to edit or delete their published content.

Comments & Likes UI: Build a separate screen or section to display and add comments for a specific post, and implement the interactive "Like" functionality.

UI/UX Design: Take charge of the overall design, colors, backgrounds, and screen space utilization to ensure a polished look and avoid grade deductions.

Content Tests: Write Jest unit tests for all backend routes related to posts, comments, and likes.
